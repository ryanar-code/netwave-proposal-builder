# Statement of Work
## TechStart Solutions - Website Redesign Project

**Date:** January 15, 2026
**Client:** TechStart Solutions Inc.
**Project Duration:** 12 weeks
**Total Budget:** $45,000

---

## 1. Project Overview

TechStart Solutions requires a complete website redesign to better reflect their evolution from a startup to an established B2B SaaS platform. The new website will emphasize credibility, showcase enterprise clients, and improve conversion rates for demo requests.

## 2. Scope of Work

### Phase 1: Discovery & Strategy (Weeks 1-2)
- Stakeholder interviews (3 sessions)
- Competitive analysis (5 competitors)
- User research and persona development
- Content audit of existing site
- SEO audit and keyword research
- Site architecture and navigation planning
- **Deliverable:** Strategy deck and sitemap

### Phase 2: Design (Weeks 3-6)
- Wireframes for all page templates (8 templates)
- Design system creation (colors, typography, components)
- High-fidelity mockups for desktop (8 pages)
- High-fidelity mockups for mobile (8 pages)
- Interactive prototype
- **Deliverable:** Complete design files in Figma + prototype

### Phase 3: Development (Weeks 7-10)
- Frontend development (React/Next.js)
- CMS integration (Contentful)
- Contact form and demo request functionality
- Blog template development
- SEO optimization implementation
- Performance optimization
- Accessibility compliance (WCAG 2.1 AA)
- **Deliverable:** Staging site for review

### Phase 4: Testing & Launch (Weeks 11-12)
- Cross-browser testing (Chrome, Safari, Firefox, Edge)
- Mobile device testing (iOS, Android)
- Load testing and performance verification
- Content migration from old site
- Final client review and revisions
- Production deployment
- DNS cutover
- Post-launch monitoring (2 weeks)
- **Deliverable:** Live website + documentation

## 3. Out of Scope

- Custom backend development
- E-commerce functionality
- Third-party API integrations beyond CMS
- Video production or photography
- Ongoing maintenance beyond 2-week post-launch period
- Content writing (client provides all copy)
